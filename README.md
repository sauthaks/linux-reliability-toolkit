# Linux Reliability Toolkit

A collection of Bash and Python scripts for basic system reliability monitoring.

## Features

- System health monitoring
- CPU load tracking
- Memory usage monitoring
- Disk usage monitoring
- Network checks
- Multi-server health checks
- Real-time monitoring with Python

## Scripts

system_monitor.sh
Checks CPU load, memory statistics and disk usage.

server_check.sh
Checks if important services or servers are reachable.

network_check.sh
Verifies network connectivity.

multi_server_check.sh
Runs health checks across multiple servers.

realtime_monitor.py
Python script for real-time monitoring.

## Use Case

This toolkit helps system administrators and SRE engineers quickly diagnose system health issues.
